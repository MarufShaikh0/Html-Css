Gradient

CSS Gradient হলো color transition—এক রঙ থেকে আরেক রঙে ধীরে ধীরে পরিবর্তন।এটি সাধারণত background-image প্রপার্টির মাধ্যমে কাজ করে।

Gradient-এর ধরন ৩ রকম:
Linear Gradient → এক লাইনে (উপর থেকে নিচ, বাম থেকে ডানে ইত্যাদি)।
Radial Gradient → মাঝখান থেকে চারপাশে ছড়ায়।
Conic Gradient → ঘড়ির কাটার মতো ঘুরে ঘুরে রঙ বদলায়।
