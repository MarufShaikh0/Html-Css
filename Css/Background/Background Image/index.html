Backgrond Image

background Size :

contain: পুরো ছবি দেখাও, প্রয়োজনে খালি জায়গা রাখো।

background-size: cover; দিলে parent-div এর 100% width+Height পাবে ।

100% → background image এর প্রস্থ (width) হবে তার container-এর 100%।
80% → background image এর উচ্চতা (height) হবে তার container-এর 80%।
background-size: 100px 500px; 1=width/2=Height

Multiple Image

Link Set : background-image: url('image1.png'), url('image2.jpg'), url('image3.gif'); এখানে `image1.png` থাকবে সবার উপরে, তার নিচে `image2.jpg`, এবং সবার নিচে `image3.gif`।

repeat : background-image: url('image1.png'), url('image2.jpg');background-repeat: no-repeat, repeat-x; /* image1 রিপিট হবে না, image2 অনুভূমিকভাবে রিপিট হবে */

Position : background-image: url('image1.png'), url('image2.jpg');background-position: center, bottom left; /* image1 মাঝখানে, image2 নিচে বামে */

Size : background-image: url('image1.png'), url('image2.jpg');background-size: contain, 50% auto; /* image1 contain হবে, image2 প্রস্থের 50% নেবে উচ্চতা auto */
