grid-template-Columns

grid-template-columns: 1fr 1fr 1fr; মানে ৩টি সমান প্রস্থের কলাম।

grid-template-columns: repeat(3, 1fr); মানেও একই, ৩টি সমান প্রস্থের কলাম তৈরি হবে।

grid-template-columns: repeat(3, minmax(0, 1fr)); মানে ৩টি কলাম থাকবে, প্রতিটি কলাম সমান জায়গা নেবে কিন্তু প্রয়োজন অনুযায়ী ০ প্রস্থ পর্যন্ত ছোট হতে পারবে।

responsive Grid

ফ্লেক্সিবল ইউনিট (fr, %, auto):

fr (fraction): এটি গ্রিড কন্টেইনারের "উপলব্ধ স্থান" (available space) এর একটি অংশকে বোঝায়। যেমন 1fr 1fr 1fr দিলে ৩টি কলাম সমান স্থান নেবে। স্ক্রিন ছোট হলে কলামগুলোও ছোট হবে, বড় হলে বড় হবে, কিন্তু সমানুপাতিক হার বজায় রাখবে।
% (শতাংশ): কলাম বা সারির আকার প্যারেন্ট এলিমেন্টের আকারের শতাংশ হিসেবে নির্ধারণ করে।
auto: কন্টেন্টের আকার অনুযায়ী উচ্চতা বা প্রস্থ স্বয়ংক্রিয়ভাবে সেট হয়।
