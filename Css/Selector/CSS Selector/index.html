Css Selector

Selector List : Tag Selector / Class Selector / ID Selector / Universal Selector

Tag Selector :

সরাসরি tag এর নাম ধরে CSS Call দিতে হয় । Ex : p-color:red

Class Selector :

সরাসরি Class এর নাম ধরে CSS Call দিতে হয় । Ex : .class-color:red

id Selector :

সরাসরি ID এর নাম ধরে CSS Call দিতে হয় । Ex : #id-color:red
